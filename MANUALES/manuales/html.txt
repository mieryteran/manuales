	scheme://prefix.domain:port/path/filename 

	
    esquema - define el tipo de servicio de Internet (lo más común es HTTP o HTTPS )

    Dominio - define a Internet de nombres de dominio (como w3schools.com)

    ruta - define una ruta en el servidor (Si se omite: el directorio raíz del sitio)
    nombre de archivo - define el nombre de un documento o recurso

	esquemas
